# 🛢️ Log Converter - تحويل ملفات LIS/DLIS لـ LAS/DLIS

تطبيق ويب بسيط لتحويل ملفات الوج (LIS و DLIS) إلى صيغ LAS أو DLIS باستخدام Streamlit.

## ✨ المميزات
- رفع ملفات LIS أو DLIS
- تحويل لـ LAS (نصي) أو DLIS (ثنائي)
- معاينة البيانات قبل التحميل
- واجهة سهلة وبسيطة

## 🚀 التشغيل المحلي
```bash
# 1. تثبيت المتطلبات
pip install -r requirements.txt

# 2. تشغيل التطبيق
streamlit run app.py
